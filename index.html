    MrHelper¶

    • MrHelper：Medical Reriew Helper
    • 是一个用于整理标注参考文献、快速创建医学综述初稿的WORD文档及ENDNOTE参考文献生成器。

    特色¶

    • 使用Web Of Science, Pubmed, CNKI, 万方数据库文献导出文件
    • 便捷查看文献网页、全文、是否核心/SCI文献
    • CS/LCS/CR/LCR 相关文献显示
    • 导出生成含有ENDNOTE临时引文的WORD综述初稿

    使用方法¶

    • 下载 - 64位
    • 快速入门 - 链接
    • 帮助文档 - 知乎（草稿）
    • 解压后执行 mrhelper.exe

    源码执行：¶

    • mrhelper.py
    • python 3.6
    • python-docx 
      pip install python-docx
    • PyQt5 5.10.1 
      pip install PyQt5==5.10.1
    • Beautiful Soup 
      pip install beautifulsoup4
    • defusedxml 
      pip install defusedxml
    • requests 
      pip install requests
    • requests[socks] 
      pip install requests[socks]
    • pyinstaller 
      pip install pyinstaller

    依赖库¶

    • Python - 编程语言
    • python-docx - 生成综述WORD文稿
    • PyQt5 - (GPL) 程序界面
    • Beautiful Soup 解析抓取的文献页面
    • defusedxml - 安全解析CNKI E-Study .eln格式
    • requests/requests[socks] - Pubmed, CNKI, 万方文献页面抓取
    • Pyinstaller - 打包可执行程序文件（.exe）
